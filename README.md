# Blaze Burn Restaurant Landing Page

A stunning, modern landing page for **Blaze Burn** - Ahmedabad's premium all-vegetarian, multi-themed food destination.

![Blaze Burn Hero](/home/odoo/.gemini/antigravity/brain/2ccbfc83-30cc-4c50-a32a-15fb912534ca/hero_section_1770050304819.png)

## 🚀 Tech Stack

- **Frontend**: React 18 + Vite
- **Backend**: Node.js + Express
- **Styling**: Vanilla CSS with custom design system
- **Typography**: Google Fonts (Playfair Display + Inter)

## ✨ Features

- 🎨 **Premium Design**: Vibrant food-inspired colors and elegant typography
- ⚡ **Smooth Animations**: Fade-ins, hover effects, and micro-interactions
- 📱 **Fully Responsive**: Optimized for desktop, tablet, and mobile
- 🔍 **SEO Optimized**: Meta tags and semantic HTML
- 🎯 **Six Sections**: Hero, About, Menu (300+ items), Features, Contact, Footer
- 💼 **Investor-Ready**: Professional presentation perfect for pitching

## 🏃 Quick Start

### Frontend (React)

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The app will open at `http://localhost:5173`

### Backend (Node.js)

```bash
# Navigate to server directory
cd server

# Install server dependencies
npm install

# Start the server
npm start
```

The server runs on `http://localhost:3000`
